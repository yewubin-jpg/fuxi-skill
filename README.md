# 伏羲.skill — 专家级认知操作系统生成器

> 「仰观象于天，俯观法于地。去伪存真，提炼精华。」

伏羲.skill 是一个元级 Skill（Meta-Skill），用于将任意人物的最优精华自动提炼为可供大众直接使用的**专家级认知操作系统 Skill**。

## 六爻画卦方法论

通过 6 个 Agent 并行调研，从 6 个正面维度深度提炼人物精华：

| 维度 | 名称 | 提炼内容 |
|------|------|----------|
| 太极 | 核心道法 | 最核心的商业哲学与底层逻辑 |
| 两仪 | 决策与行动 | 关键时刻的正确决策与成功案例 |
| 四象 | 表达与魅力 | 经典金句、演讲风格与个人魅力 |
| 八卦 | 社会正向价值 | 行业正面评价与社会积极贡献 |
| 天干 | 对谈与真我 | 访谈中最真诚、最有魅力的瞬间 |
| 地支 | 成长与传承 | 蜕变故事、师承关系与正面影响力 |

## 使用方式

在 Manus 中说：「提炼一个乔布斯的精华」「帮我创建一个马斯克的商业版Skill」「蒸馏叶武滨的最优一面」即可触发。
